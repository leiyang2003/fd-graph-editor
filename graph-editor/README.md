# 有向图编辑器 (Graph Editor)

网页版有向图编辑器，对应 `Run.py` 中的故事流程图，可在浏览器中编辑节点与边并导出 JSON。

## 使用方式

1. **直接打开**：用浏览器打开 `index.html`（双击或在终端执行 `open index.html`）。
2. **本地服务器**（推荐，避免部分浏览器对 file:// 的限制）：
   ```bash
   cd /Users/leiyang/Desktop/Coding/DSL/graph-editor
   python3 -m http.server 8080
   ```
   然后访问 http://localhost:8080

## 功能

- **＋ 添加节点**：输入节点 ID、显示标签、颜色。
- **⇢ 连接两节点**：输入起点 ID、终点 ID、边上标签（可选）。
- **✎ 编辑节点**：选中一个节点后点击，可改标签和颜色。
- **删除**：选中节点或边后点击「删除 节点/边」。
- **加载示例图（古镜）**：载入与 `Run.py` 一致的示例故事图。
- **导出 JSON**：下载当前图为 `graph.json`，便于备份或给其他程序用。
- **导入 JSON**：从本地上传 `graph.json` 恢复或合并图。

画布支持拖拽节点、缩放、平移；多选后可批量删除。
